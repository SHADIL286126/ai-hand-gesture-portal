# AI Hand Gesture Recognition Portal

## Overview

A Django web application that performs real-time hand gesture recognition using OpenCV and MediaPipe and allows users to play Rock-Paper-Scissors against an AI opponent.

## Features

- User Authentication
- Real-Time Webcam Capture
- Hand Gesture Recognition
- Rock-Paper-Scissors AI Gameplay
- Match History
- Dashboard Analytics
- Leaderboard

## Technologies Used

- Python
- Django
- OpenCV
- MediaPipe
- JavaScript
- HTML/CSS
- SQLite
